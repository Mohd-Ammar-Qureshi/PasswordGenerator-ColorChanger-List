# React Native Mini Projects 🚀

This repository contains 3 beginner-friendly React Native projects created for learning core React Native concepts like state management, styling, lists, user interaction, and performance optimization.

---

# 📱 Projects Included

## 1. Password Generator 🔐

A simple password generator app that creates secure random passwords based on user preferences.

### Features
- Generate random passwords
- Custom password length
- Strong password generation
- Simple and clean UI

### Concepts Used
- `useState`
- Random string generation
- TextInput handling
- Button interactions
- Styling with StyleSheet

---

## 2. Random Color Generator 🎨

An app that changes the background color randomly when the button is pressed.

### Features
- Generate random RGB/HEX colors
- Change UI color dynamically
- Real-time background updates

### Concepts Used
- `useState`
- Dynamic styling
- Button press handling
- Random number generation

---

## 3. Random Color List using FlatList 📋

A performance-focused app that renders a large list of random colors using `FlatList`.

### Features
- Generate 1000 random colors
- Smooth scrolling performance
- Optimized list rendering
- Color preview boxes

### Concepts Used
- `FlatList`
- Large list optimization
- Dynamic rendering
- React Native performance basics
- Reusable components

---

- FlatList
- StyleSheet API
